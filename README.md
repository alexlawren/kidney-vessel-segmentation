# Программный модуль локализации микрососудистого русла на гистологических срезах почек на базе нейросети U-Net

Репозиторий содержит материалы производственной практики по разработке программного модуля автоматической сегментации микрососудистых структур (капилляров, артериол, венул) на гистологических изображениях тканей почек человека.

## 🛠 Технологический стек
* **Язык программирования:** Python 3.10+
* **Фреймворк глубокого обучения:** PyTorch
* **Библиотека моделей сегментации:** `segmentation_models_pytorch` (SMP)
* **Аугментация данных:** `Albumentations`
* **Обработка изображений и визуализация:** Pillow, Matplotlib, OpenCV
* **Интерфейс демонстрации:** Streamlit *(в разработке)*
